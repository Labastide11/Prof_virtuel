Le verbe

Je sais repérer le verbe et retrouver sa forme à l’infinitif.

🦉
1Je retiens
Le verbe indique souvent une action ou un état. Il change quand le temps ou le sujet change.
2Exemple
Les élèves jouent. → jouer
Mila est contente. → être
3🦉 L’astuce de Maître Hibou
Change le temps : Aujourd’hui il chante. → Hier il chantait. Le mot qui change est le verbe.
4Tu as compris ?
Quel est le verbe dans : Le chien court dans le jardin.
